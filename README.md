## 📁 هيكلة المستودع

تم تنظيم البيانات بطريقة بسيطة وقابلة للتوسّع:


/tafsir
├── tafsir-al-qurtubi
├── tafsir-al-suddi
├── tafsir-tanwir-al-miqqbas
├── tafsir-al-baghawi
├── tafsir-al-tabari
├── tafsir-al-wasit
├── tafsir-ibn-kathir
└── tafsir-al-muyassar


كل مجلد يحتوي على بيانات بصيغة JSON تمثل:

* تفسير السور
* تفسير الآيات آية بآية
* صيغة منظمة قابلة للقراءة البرمجية والاستخدام عبر API



نرحب بالمساهمات لتحسين المشروع عبر:

* تحسين هيكلة البيانات  
* إضافة مصادر تفسير جديدة  
* تصحيح الأخطاء أو التناقضات  
* تحسين التوثيق
